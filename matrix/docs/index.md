# 欢迎来到 MetroWind 的矩阵服务器

Matrix 是一种即时聊天服务。它和电子邮件很像：任何人都可以运营自己的服务器，不同服务器上的人仍然可以互相发消息、加入同一个群聊。你的账号在 `darksair.org`，但也可以和使用其他 Matrix 服务器的朋友聊天。你还可以自己选择用哪一个聊天软件登录；无论在手机上用 FluffyChat，还是在电脑浏览器里用 Cinny，登录同一个账号后都可以进入同样的聊天和群组。

这是朋友们使用的 Matrix 服务器。你只需要安装一个聊天软件、登录账号，就可以加入群聊；不用理解服务器怎样运作。

## 选择聊天软件

- **移动端：**[FluffyChat](https://fluffychat.im/)。Android、iPhone 和 iPad 都可以使用；推荐用于日常聊天，界面简单，图片、文件和贴纸都能正常使用。
- **网页／电脑：**[Cinny](https://app.cinny.in/)。不需要安装软件，在浏览器打开即可。请不要在公共电脑上保持登录。

请先阅读下面的登录步骤。最容易填错的是服务器地址和用户名：两者不是同一个东西。

## 先记住这两个信息

> **服务器（Homeserver）：** `darksair.org`
>
> **用户名：** `@你的用户名:darksair.org`

例如，用户名是 `alice` 时，应填写 `@alice:darksair.org`。不要把服务器地址写进用户名，也不要漏掉开头的 `@` 和后面的 `:darksair.org`。

## 第一次登录

1. 安装并打开 FluffyChat，选择“登录已有账号”或 “Sign in”。不要选择注册新账号。
2. 软件询问 Homeserver（服务器）时，填写 `darksair.org`。
3. 按上面的完整格式填写用户名，例如 `@alice:darksair.org`。
4. 填写管理员发给你的初始密码。
5. 软件可能会要求设置一个 passcode、恢复口令或安全备份口令。这和登录密码是两个不同的东西。现在先选定一个只有你知道的新密码，并把它用作这个 passcode；下一步也会把账户密码改成同一个值，这样以后只需要记住一个密码。请不要把它发给别人。
6. 进入聊天软件后，立刻在设置中把**账户密码**改成只有你知道的新密码。
   1. 打开菜单

   2. 打开设置
      ![password 1](assets/imgs/password-1.png){.screenshot}

7. 告诉管理员你已登录；管理员会把你邀请进群。

## 网页端 Cinny

在电脑浏览器打开 [Cinny](https://app.cinny.in/)，同样填写上面的服务器、完整用户名和密码即可。浏览器的无痕窗口或清理网站数据后，往往需要重新登录；自己的电脑上使用普通窗口会更方便。

## 换手机或添加电脑

在新设备上用同一账号登录。软件可能会要求你在旧设备上确认，或输入之前设置的恢复口令；这是正常的安全确认。旧设备已经不可用、忘记账户密码，或找不到恢复口令时，不要反复猜测，直接联系管理员。

## 需要帮助？

登录、换设备或忘记密码时，直接联系管理员即可。
